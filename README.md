# JIRA Clippy by ucpwang
![JIRA ](src/jira-clippy-logo-64.png)
Copy jira issues to your clipboard as 'PROJECT-777 - Description(URL)'.

## Build
To build the extension use:
```
chmod +x build.sh
./build.sh
```

## install
1. go to `chrome://extensions`
2. 확장 프로그램 압축
3. 압축해제된 확장 프로그램 로드
